## 실행 절차 (장별 반복)
1. `/tmp/pdfcheck/추출 텍스트`에서 해당 표준 교재 장 페이지 범위 grep → 본문·의사코드·증명 추출.
2. content_algoN.js 뼈대를 viz(code+build+draw) 또는 concept:true로 전환.
3. 신규 깊이 분기를 content_algo_br.js({id,branchOf,code/build/draw 또는 concept})+algo_br.json(한·영)로 추가.
4. algo.html 리로드 → 스텝/코드 동기·concept 패널 검증 + 스크린샷 + 콘솔에러0 + 수학회귀0.
5. 커밋 → push(라이브 갱신).
