# 알고리즘 진행 순서 (SSOT) — 수학 트랙과 동일 morph 방식

> 수학(math.html)과 같은 엔진(engine.js)·5원칙(예고↔회수)·한영병기·화면번호·학습패널을 그대로 사용.
> ★강점: **수학 트랙을 적극 회수** — 가우스 합(13장)→O(1), 로그(7장)→O(log n), 함수 성장(5·7장)→복잡도 곡선,
> 피보나치(13장 수열)→DP, 그래프(6장 좌표)→네트워크.
> 진행축: 알고리즘·복잡도 → 자료구조 → 정렬 → 탐색 → 트리 → 그래프 → 동적계획법 → 설계 패러다임.

## 제1장 알고리즘과 복잡도
- 1.1 알고리즘이란 (명확한 단계·입출력·유한성)
- 1.2 효율성 (같은 답, 다른 비용)
- 1.3 빅오 표기법 (O(1)·O(log n)·O(n)·O(n²)·O(2ⁿ))

## 제2장 자료구조
- 2.1 배열 (인덱스 접근 O(1))
- 2.2 연결 리스트 (포인터)
- 2.3 스택과 큐 (LIFO·FIFO)
- 2.4 해시 테이블 (O(1) 조회)

## 제3장 정렬
- 3.1 기초 정렬 (버블·선택·삽입, O(n²))
- 3.2 병합 정렬 (분할정복, O(n log n))
- 3.3 퀵 정렬 (피벗 분할)

## 제4장 탐색
- 4.1 선형 탐색 (O(n))
- 4.2 이분 탐색 (O(log n), 정렬 전제)

## 제5장 트리
- 5.1 이진 트리·순회
- 5.2 이진 탐색 트리 (BST)
- 5.3 힙과 우선순위 큐

## 제6장 그래프
- 6.1 그래프 표현 (인접 행렬·리스트)
- 6.2 너비 우선(BFS)·깊이 우선(DFS)
- 6.3 최단 경로 (다익스트라)

## 제7장 동적 계획법
- 7.1 메모이제이션 (피보나치)
- 7.2 최적 부분 구조 (배낭·LCS)

## 제8장 알고리즘 설계 패러다임
- 8.1 그리디
- 8.2 분할 정복
- 8.3 계산의 한계 (P vs NP) + 종합 피날레
